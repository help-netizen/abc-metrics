# Импорт данных через API

## Цель

Запустить синхронизацию данных из Workiz API для заполнения пустых таблиц fact_jobs, fact_leads, fact_payments.

## Действия

1. Запустить синхронизацию Jobs через `POST /api/test/workiz/jobs/sync-full`
2. Запустить синхронизацию Leads через `POST /api/test/workiz/leads/sync-full`
3. Проверить результаты синхронизации (количество записей в таблицах)
4. При необходимости добавить endpoint для Payments и запустить синхронизацию

## Ожидаемый результат

- fact_jobs: заполнена данными за последние 30 дней
- fact_leads: заполнена данными за последние 30 дней
- fact_payments: можно синхронизировать после добавления endpoint (опционально)
